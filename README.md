# 🔢 Menu Driven Data Structure Program (Python)

## 📌 Description
This project is a menu-driven Python program that demonstrates built-in functions of different data structures such as List, Dictionary, Tuple, Set, and String.

The program is divided into two files:
- One file contains all functions
- Another file contains the main menu and function calls

---

## 📂 Project Structure
```
main.py
Menudrivenofallbuiltinfunctionofdatastructure2.py
```

---

## 🚀 Features
- Menu-driven program
- Demonstrates built-in functions of:
  - List
  - Dictionary
  - Tuple
  - Set
  - String
- Modular programming (separate files)
- Error handling using try/except

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open terminal in project folder  
3. Run:

```bash
python main.py
```
