# 🔢 Menu Driven Data Structure Program (Python)

## 📌 Description
This project is a menu-driven Python application designed to demonstrate the use of various built-in functions available in different data structures such as List, Dictionary, Tuple, Set, and String.

The program allows the user to interactively select a data structure from the main menu and then explore multiple built-in functions related to that data structure through a sub-menu system.

Each data structure has its own dedicated function, where users can choose different operations and observe how the built-in methods work with real-time examples.

The project follows a modular programming approach by separating the main menu logic and the function implementations into different files, making the code clean, reusable, and easy to understand.

---

## 🎯 Objectives
- To understand Python built-in functions for different data structures
- To implement a menu-driven program using loops and conditionals
- To practice modular programming using multiple files
